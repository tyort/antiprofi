# Deploy Guide

Краткая памятка по продовому VPS и первичному деплою проекта `antiprofi`.

## VPS

- Провайдер: `Spaceweb`
- Hostname: `vps-tyort_vps`
- IP: `168.222.202.149`
- ОС: `Ubuntu 26.04 LTS`
- Тариф: `2 CPU / 2 GB RAM / 15 GB NVMe`
- Дата создания: `2026-06-29`
- Доступ: `SSH` по ключам

## Что хранить отдельно

Секреты и чувствительные данные не хранить в репозитории:

- пароли
- приватные `SSH`-ключи
- значения переменных окружения
- доступы к почте, БД и сторонним сервисам

## Подготовка к деплою

1. Убедиться, что на сервере установлены `Docker` и `Docker Compose`.
2. Создать `.env` в корне проекта и заполнить нужные переменные окружения.
3. Проверить, что домен указывает на `IP` сервера.
4. Открыть только нужные порты для приложения, `SSH` и `HTTPS`.

## Первый запуск

```bash
docker compose up -d --build
```

После сборки приложение должно быть доступно на сервере.

## Обновление

При изменениях в коде или окружении:

```bash
docker compose down
docker compose up -d --build
```

## Проверка

- Посмотреть логи: `docker compose logs -f web`
- Остановить контейнеры: `docker compose down`
- Проверить, что сервис отвечает по домену или `IP`
